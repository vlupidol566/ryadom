# Сценарии использования приложения

## Участники

- **Ведущий** — зрячий помощник: видит карту и видео ведомого, направляет по маршруту.
- **Ведомый** — человек, которого ведут (в т.ч. **слепой или слабовидящий**): передаёт видео с камеры и (в перспективе) геолокацию.

---

## Сценарий 1: Слепой (ведомый)

### Цель
Получить помощь по маршруту: ведущий видит «картинку» с камеры и может подсказывать голосом (по телефону или позже по приложению).

### Шаги

1. **Открыть приложение**
   - TalkBack: «Выбор роли. Ведущий. Ведомый. Выбрать, кому звонить.»

2. **Выбрать роль «Ведомый»**
   - Нажать карточку «Ведомый» («Передача видео и геолокации»).
   - Открывается экран **«Выберите ведущего»** — список контактов.

3. **Выбрать ведущего**
   - Нажать на контакт ведущего (например, «Мама») — переход на экран «Ведомый» с подписью «Подключение к Мама».
   - Либо нажать иконку «Позвонить» у контакта — звонок, потом вернуться и нажать на контакт для подключения.

4. **Подключиться**
   - На экране «Ведомый» нажать «Связаться с ведущим».
   - Озвучка: «Подключение…», затем при успехе можно сообщить «Подключено» (если добавим).
   - Камера и геолокация передаются ведущему.

5. **Дальше**
   - Слепой идёт по маршруту, ведущий смотрит видео/карту и подсказывает по телефону (отдельный голосовой звонок) или позже — голосом через приложение.

### Что важно для слепого
- Все элементы с понятными `contentDescription` (уже частично есть).
- Один логичный порядок: «сначала контакты/звонок → потом ведомый → связаться».
- Крупные зоны нажатия, без «визуальных только» подсказок.
- После подключения — чёткое озвучивание состояния («Подключено» / «Нет связи»).

## Сценарий 2: Ведущий

### Цель
Увидеть видео и (позже) место ведомого на карте и направлять его голосом.

### Шаги

1. **Открыть приложение**
   - Выбрать карточку «Ведущий» («Управление маршрутом и видео»).

2. **Экран «Ведущий»**
   - В шапке есть кнопка **«Позвонить ведомому»** — открывает контакты, можно позвонить и сказать: «Открой приложение, выбери Ведомый, выбери меня в списке и нажми Связаться с ведущим.»
   - Либо ведущий просто ждёт подключения.

3. **Контент экрана**
   - Сверху: блок с видео ведомого (пока пусто: «Ожидание подключения ведомого»).
   - Снизу: карта (Яндекс.Карты).
   - Когда ведомый нажимает «Связаться с ведущим», в блоке появляется видео, карта при необходимости может показывать точку ведомого (после реализации геолокации).

4. **Направление**
   - Ведущий смотрит на видео и карту и подсказывает ведомому по телефону (голосовой звонок) или позже — голосом/чат в приложении.

### Что важно для ведущего
- Сразу видно, подключён ли ведомый (видео есть/нет).
- Карта и видео на одном экране, без лишних шагов.
- При желании — быстрый переход в контакты, чтобы позвонить ведомому (иконка в шапке или отдельная кнопка «Позвонить ведомому»).

## Возможные улучшения сценария

### Для слепого (ведомый)
1. **«Позвонить и подключиться»**  
   После выбора «Ведомый» — экран «Кому подключаться?»: список контактов, выбор контакта → сначала звонок (опционально), затем одна кнопка «Подключиться к ведущему». Так явно связываем «кому звонить» и «к кому подключаться».

2. **Озвучка состояния**  
   При подключении/обрыве — TalkBack или короткий звук: «Подключено» / «Соединение разорвано».

3. **Минимум шагов**  
   Текущий порядок «Главная → Ведомый → Связаться» уже простой; при добавлении «выбора ведущего из контактов» важно не раздувать количество экранов.

### Для ведущего
1. **Быстрый звонок ведомому**  
   На экране «Ведущий» — кнопка/иконка «Позвонить ведомому»: открывает контакты или последний выбранный контакт и сразу предлагает позвонить. Тогда сценарий: «Открыл ведущий → нажал „Позвонить ведомому“ → сказал открыть приложение → ждёт подключения».

2. **Геолокация на карте**  
   Пока только видео; когда будет передача координат ведомого — показывать точку/маршрут на карте ведущего.

3. **Один ведущий — один ведомый**  
   Текущая логика «один ведущий, один ведомый» на сервере сигналинга этому сценарию соответствует.

## Кратко по потоку

| Кто        | Действия |
|-----------|----------|
| **Слепой (ведомый)** | Открыть приложение → «Ведомый» → «Выберите ведущего» → выбрать контакт ведущего → на экране «Ведомый» нажать «Связаться с ведущим». |
| **Ведущий** | Открыть приложение → «Ведущий» → при необходимости нажать «Позвонить ведомому» в шапке → ждать подключения → смотреть видео и карту, направлять голосом. |

Реализовано: кнопка «Позвонить ведомому» на экране ведущего и экран «Выберите ведущего» перед подключением на стороне ведомого.
