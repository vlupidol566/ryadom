# Сервер сигналинга WebRTC

Устанавливается на **удалённый компьютер**, который всегда в интернете. Телефоны подключаются к нему по WebSocket.

## На удалённом компьютере

1. Установить Node.js (если ещё нет): https://nodejs.org/

2. Скопировать папку `signaling-server` на этот компьютер.

3. В папке выполнить:
   ```bash
   npm install
   npm start
   ```

4. Открыть порт **9090** в файрволе и/или на роутере (проброс порта), чтобы снаружи был доступ к этому порту.

5. Узнать внешний IP удалённого компьютера (или настроить домен).  
   Телефоны будут подключаться по адресу:
   - `ws://ВАШ_IP:9090`  
   или, если настроите HTTPS и WSS:
   - `wss://ваш-домен.ru`

## Переменные окружения

- `PORT` — порт (по умолчанию 9090):  
  `PORT=9090 node server.js`

## В приложении на Android

В коде нужно указать этот же адрес сервера, например:

```kotlin
val signalingUrl = "ws://ВАШ_IP:9090"
```

Или с доменом: `ws://signaling.ваш-домен.ru:9090`
